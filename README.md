# NEXUS Worlds

NEXUS Worlds is a TypeScript/Vite prototype for a living AI civilisation sim. The current slice includes:

- Planet evolution from a Day 1 settlement toward futuristic technology.
- A tech tree, building unlocks, speed modes, and activity feed.
- Controlled agent creation that can grow the population to 50 citizens per in-game day.

## Run In Git Bash

```bash
cd "/c/Users/howel/OneDrive/Documents/ready player two"

```

