# Unreal Open-World Direction

This document records the new long-term direction for NEXUS Worlds.

## New Long-Term Goal

NEXUS Worlds should eventually become an Unreal Engine open-world AI civilisation game.

The ambition is to create a more advanced open-world experience than traditional crime/open-world sandbox games by focusing on living AI society, evolving technology, player-created stories, and self-directed citizens.

GTA-style open-world scale can be used as a reference for ambition only.

Do not copy GTA branding, characters, map design, missions, vehicles, UI, radio, storylines, gangs, businesses, or protected assets.

NEXUS Worlds must remain original.

## Core Difference

The goal is not to clone GTA.

The goal is to build:

```text
A living open-world AI civilisation where every citizen has goals, memories, jobs, relationships, routines, needs and ambitions.
```

The player is the Founder/Creator with an invisible avatar and freedom to shape the world.

## Why Unreal Engine Later

Unreal Engine is the better long-term target for:

- large open worlds
- cinematic visuals
- vehicles
- advanced lighting
- character animation
- environmental detail
- city-scale simulation
- high-quality weather
- future console/PC style builds

## Why Not Jump Too Fast

A GTA-scale game is enormous.

The safe route is:

1. prove the gameplay loop
2. build the simulation systems
3. create a small Unreal vertical slice
4. expand gradually

Do not try to build a full city first.

## Correct Build Path

### Phase 1: Browser Prototype

Use current React/Convex project to prove:

- player role
- agents
- world events
- activity feed
- tech tree
- Advance Day
- agent goals
- storm demo

### Phase 2: Unreal Vertical Slice

Build a small playable Unreal map:

- one settlement
- six AI citizens
- campfire
- shelter project
- resource piles
- storm event
- simple vehicle placeholder later
- basic third-person/free-camera movement

### Phase 3: Unreal Simulation Connection

Connect Unreal client to backend world state:

- Convex or future backend stores simulation
- Unreal renders the world
- AI planner proposes actions
- validation checks rules
- backend updates world
- Unreal receives updates

### Phase 4: Living Open World

Expand into:

- larger settlement
- roads
- vehicles
- shops
- jobs
- factions
- quests
- economy
- crime/risk systems if original
- citizens travelling
- technology progression
- flying cars
- spaceport

## Better Than GTA Angle

NEXUS Worlds can beat traditional open-world games in one specific way:

```text
The world is not only scripted. It thinks, remembers and evolves.
```

Focus advantages:

- AI citizens make their own decisions
- events change society
- technology evolves from problems
- buildings are constructed because agents need them
- relationships affect behaviour
- quests emerge from the world
- player choices create new history
- world keeps a memory timeline

- interaction system
- agent actors
- agent nameplates

- dialogue bubbles

## Do Not Build Yet

Do not start with:

- full GTA-scale city
- multiplayer
- realistic traffic
- police/crime simulation
- massive vehicle catalogue
- advanced economy
- online marketplace
- VR

Build the small slice first.
