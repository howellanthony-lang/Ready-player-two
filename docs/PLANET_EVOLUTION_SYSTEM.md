# Planet Evolution System

The Planet Evolution System allows a world to grow from a basic Day 1 settlement into a futuristic civilisation.

## Purpose

The goal is to make each world feel alive, progressing from survival to advanced technology.

The system should allow AI agents to:

- Gather resources
- Build structures
- Research technology
- Create jobs
- Form society
- Unlock new ages
- Build futuristic systems
- Eventually create flying cars, spaceports and advanced cities

## Civilisation Ages

1. Origin Age
2. Settlement Age
3. Town Age
4. Industrial Age
5. Electrical Age
6. Digital Age
7. Futuristic Age

## Speed Modes

- Realistic Mode: 1 real minute = 1 in-game hour
- Fast Mode: 1 real minute = 1 in-game day
- Founder Mode: 10 seconds = 1 in-game day
- God Mode: 1 second = 1 in-game day

## Core Progress Values

Each world tracks:

- Knowledge points
- Infrastructure points
- Culture points
- Economy points
- Energy points
- Transport points
- Research points

## Agent Involvement

Agents should influence evolution through their roles.

Examples:

- Builders create infrastructure points
- Farmers create food stability
- Teachers create knowledge points
- Scientists create research points
- Engineers create energy and transport points
- Leaders create culture and economy points

## Key Rule

The agents should appear to drive civilisation forward.

The game system controls valid unlocks, but AI conversations and goals should explain why the society evolves.
