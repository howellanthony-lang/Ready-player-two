# Ready Player Two Roadmap

## Phase 0: Foundation

- Import AI Town foundation.
- Confirm local setup works.
- Keep the 3D viewport rendering.
- Keep `npm run dev` working.

## Phase 1: First Playable MVP

- Visible 3D world.
- WASD player movement.
- Six starter citizens.
- Click-to-inspect citizens.
- Activity feed.
- Civilisation panel.
- Local world event input.

## Phase 2: Better Simulation

- Agent needs.
- Agent jobs.
- Daily schedules.
- Relationship changes.
- Memory importance scoring.
- Basic teamwork and conflict.

## Phase 3: AI Conversations

- Mock conversation provider first.
- Model router interface.
- Ollama/OpenAI providers later.
- Speech bubbles.
- Conversation summaries.
- Memory updates from conversations.

## Phase 4: Persistent World

- Move world state into Convex.
- Persist agents, buildings, memories, feed entries, technologies, and events.
- Add save/load/reset flows.

## Phase 5: Creator Tools

- Custom character creator.
- Custom world template.
- Export story log.
- Shareable world file.

## Phase 6: Commercial Version

- User accounts.
- Save slots.
- Paid world packs.
- Education mode.
- Streamer mode.
- Marketplace for characters/world templates.

## Phase 7: Advanced Version

- Multiplayer presence.
- Voice conversations.
- AI-generated quests.
- Agent economy.
- Procedural buildings.
- Flying vehicles.
- Spaceport travel.
- Mod support.
